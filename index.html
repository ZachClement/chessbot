Chessbot

Chess.com bot as Google Chrome extension

View on GitHub View at Chrome Store

Introducing.

This automated computer program worked as Google Chrome extension. You can install them into your browser, then go to [Chess.com](http://chess.com) log in or register new account and play.

How to play

When you start playing new game (online or live mode) you will see a robot icon ![Robot icon](https://raw.githubusercontent.com/recoders/chessbot/master/images/robot-20.png) (defers on playing mode). Right from them you will see best next move.

Legal information

Use cheating program when playing is illegal. See there: http://www.chess.com/legal#rules
